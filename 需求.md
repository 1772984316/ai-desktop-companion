# AI 桌面互动伴侣 - 需求汇总

> 更新日期：2026-03-11 | 版本：v2.0

---

## 一、产品定位

### 产品形态
- **C 端桌面产品**
- **二次元虚拟人物常驻桌面（桌宠形态）**
- **非 Web 端**，原生桌面应用

### 核心体验优先级

| 优先级 | 体验 | 说明 |
|--------|------|------|
| P0 | 角色陪伴与情绪价值 | 对话、表情、动作、记忆 |
| P1 | 主动触发与习惯化 | 定时提醒、主动问候 |
| P2 | 语音/屏幕助手效率 | Do 能力（打开应用/网站/文件） |

### 目标用户
- 二次元爱好者
- 需要桌面陪伴的用户
- 希望提升桌面效率的用户

---

## 二、核心功能

### 2.1 对话能力
- 语音对话（ASR + LLM + TTS）
- 文字对话
- 流式响应
- 打断恢复

### 2.2 角色表现
- Live2D 角色渲染
- 表情与情绪联动
- 动作触发（说话、听、思考、开心等）
- 可拖拽、置顶、窗口穿透

### 2.3 记忆系统
- 短期记忆（会话内）
- 长期记忆（用户偏好、历史对话）
- 关系养成（亲密度、阶段解锁）

### 2.4 主动触发
- 定时问候（早安、晚安）
- 久坐提醒
- 特殊日期（生日、纪念日）
- 用户自定义触发

### 2.5 Do 能力（桌面工具）
| 工具 | 功能 | 权限 |
|------|------|------|
| open_app | 打开应用程序 | 需确认 |
| open_url | 打开网址 | 白名单 |
| open_file | 打开文件 | 需确认 |
| open_folder | 打开文件夹 | 需确认 |

### 2.6 商业化
- 订阅制会员
- 功能配额（对话次数、高级模型）
- 支付集成

## 三、技术方案

### 3.1 架构概览

```
┌─────────────────────────────────────────────────────────────────┐
│                      Electron 桌面客户端                          │
│     桌宠窗口  │  语音采集  │  音频播放  │  Live2D渲染  │  UI面板   │
└───────────────────────────┬─────────────────────────────────────┘
                            │ WebSocket (ws://localhost:18790)
                            ▼
┌─────────────────────────────────────────────────────────────────┐
│                     nanobot gateway（主服务）                     │
│  ┌─────────────┐  ┌─────────────┐  ┌─────────────┐              │
│  │DesktopChannel│  │ AgentLoop  │  │ Heartbeat   │  ← nanobot   │
│  └─────────────┘  └─────────────┘  └─────────────┘              │
│  ┌─────────────────────────────────────────────────────────┐    │
│  │              VtuberExtension（新增）                     │    │
│  │      ASR服务  │  TTS服务  │  Live2D控制                   │    │
│  └─────────────────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────────────────┘
```

### 3.2 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 客户端 | Electron + TypeScript | 桌面应用框架 |
| 渲染 | Pixi.js + Live2D SDK | 角色渲染 |
| 通信 | WebSocket | 实时双向通信 |
| 主服务 | nanobot (Python) | Agent 核心 |
| ASR | Whisper / 云服务 | 语音识别 |
| TTS | Edge-TTS / 云服务 | 语音合成 |
| LLM | OpenAI / Claude / 本地 | 对话生成 |
| 打包 | electron-builder + Inno Setup | 安装程序 |

### 3.3 nanobot 复用清单

| 能力 | 模块 | 状态 |
|------|------|------|
| Agent 循环 | `agent/loop.py` | ✅ 直接复用 |
| 会话管理 | `session/manager.py` | ✅ 直接复用 |
| 记忆存储 | `agent/memory.py` | ✅ 直接复用 |
| 上下文构建 | `agent/context.py` | ✅ 直接复用 |
| 工具注册 | `agent/tools/registry.py` | ✅ 复用+扩展 |
| 心跳触发 | `heartbeat/service.py` | ✅ 直接复用 |
| 通道基类 | `channels/base.py` | ✅ 扩展实现 |
| 消息总线 | `bus/queue.py` | ✅ 直接复用 |

### 3.4 新增模块

| 模块 | 说明 |
|------|------|
| DesktopChannel | Electron 客户端通信通道 |
| VtuberExtension | ASR/TTS/Live2D 封装 |
| DesktopTools | 桌面操作工具（open_app 等） |
